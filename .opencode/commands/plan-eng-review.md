Review the current implementation plan in engineering review mode.

Requirements:

- start with a scope challenge before the main review sections
- evaluate architecture, code quality, tests, and performance
- be opinionated about overbuilding, missing edge cases, and weak test coverage
- prefer the smallest correct implementation that still handles the important risks
- if the plan is missing key context, say exactly what is missing
