Manage project learnings for this repo.

Supported invocations:

- `/learn` for recent learnings
- `/learn search <query>` to find related past patterns
- `/learn prune` to review stale or conflicting entries
- `/learn export` to format learnings for docs
- `/learn stats` for summary counts and confidence trends
- `/learn add` to record a new learning manually

Do not make product-code changes as part of this workflow.
