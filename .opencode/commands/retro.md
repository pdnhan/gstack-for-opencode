Run a weekly engineering retrospective for this repository.

Default window: last 7 days. Pass an argument to change it:

- `/retro 24h` — last 24 hours
- `/retro 14d` — last 14 days
- `/retro 30d` — last 30 days
- `/retro compare` — compare this period vs the prior same-length window
- `/retro compare 14d` — compare with an explicit window
- `/retro global` — cross-project retro across all AI coding tools
- `/retro global 14d` — global retro with explicit window

Analyze commit history, session patterns, contributor metrics, hotspots, and code quality signals. Save a JSON snapshot for trend tracking. Output the full retrospective to the conversation.
